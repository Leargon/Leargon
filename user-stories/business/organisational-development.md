#### USER STORY 'Basic'
**AS A** company\
**IF** \
**I WANT** to define organisational units and assign a specific user as organisational unit lead\
**SO THAT** we can reflect the flow organisation

#### USER STORY 'Actualisation'
**AS AN** organisational unit lead or admin\
**IF** \
**I WANT** to administer the organisational unit's name, type, and description\
**SO THAT** all information of the organisational unit is up-to-date

#### USER STORY 'Child entities'
**AS AN** organisational unit lead or admin\
**IF** \
**I WANT** to create subunits\
**SO THAT** I can define more details

#### USER STORY 'Crossunit teams'
**AS AN** admin\
**IF** \
**I WANT** to create crossunit teams, which spans across many organisational parent units\
**SO THAT** I can define more details

#### USER STORY 'Governance roles'
**AS AN** admin\
**IF** \
**I WANT** to assign a business owner, business steward, and technical custodian to an organisational unit\
**SO THAT** these roles are automatically resolved as the effective owner, steward, and custodian for all entities and processes in bounded contexts owned by this unit, without requiring explicit assignment on every item
