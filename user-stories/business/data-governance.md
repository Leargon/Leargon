**IF** \
**I WANT** to define business entities and assign a specific user as data owner\
**SO THAT** we can develop data governance and define terms in a ubiquitous language

**AS A** data owner or admin\
**IF** \
**I WANT** to administer the data's name, description, classifications to reflect the reality\
**SO THAT** all information of the data is up-to-date
