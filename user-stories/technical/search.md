#### USER STORY 'Search catalogue'
**AS A** logged in user\
**IF** \
**I WANT** to search across the entire catalogue by entering a free-text query that matches against entity names, process names, domain names, and org unit names in all supported locales\
**SO THAT** I can quickly navigate to any item without browsing the tree views

#### USER STORY 'Filter search results by type'
**AS A** logged in user\
**IF** search results span multiple resource types\
**I WANT** to filter search results to a specific resource type (business entity, business process, business domain, organisational unit)\
**SO THAT** I can narrow down results when I know what kind of item I am looking for

#### USER STORY 'Navigate to item from search result'
**AS A** logged in user\
**IF** a search result is shown\
**I WANT** to click on a search result and be navigated directly to the detail page of that item\
**SO THAT** I can access the item in one step from the search interface
